# Architecia 🏗️

Architecia es una web que recomienda una **arquitectura de software** según las características de un proyecto.  
El usuario responde un conjunto de preguntas y el sistema analiza las respuestas para sugerir la arquitectura más adecuada.

---

## 🚀 Cómo funciona

1. El usuario responde un formulario sobre su proyecto.
2. El sistema analiza las respuestas.
3. Se recomienda una arquitectura de software.

Ejemplos de arquitecturas recomendadas:

- Monolítica
- Arquitectura en capas
- Microservicios
- Arquitectura hexagonal
- Event-driven
